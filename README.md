
# Stardust2  
Stardust is a cross platform desktop chat application built using electron, react, and the CHATKIT API.

## Project setup

Use the package manager NPM to install Stardust2.
  
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```
